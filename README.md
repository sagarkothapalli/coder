# Attendance Tracker

A simple web application to track your attendance for multiple subjects.

## Features

*   **User Authentication:** A simple login system to protect your data.
*   **Subject Management:** Add and remove subjects to track.
*   **Attendance Tracking:** Track attended and canceled classes for each subject.
*   **Percentage Calculation:** Automatically calculates your attendance percentage.
*   **Persistent Storage:** Your data is saved in your browser's local storage, so it will be there when you come back.

## Getting Started

To run this project locally, you need to have [Node.js](https://nodejs.org/) installed.

1.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    ```

2.  **Navigate to the project directory:**
    ```bash
    cd attendance-tracker
    ```

3.  **Install the dependencies:**
    ```bash
    npm install
    ```

4.  **Start the development server:**
    ```bash
    npm start
    ```

    The application will be available at `http://localhost:3000`.
